# mirgo

# 准备

1. 新建 asset 目录
2. 下载 客户端美术资源 解压重命名为 client
3. 下载 服务端配置文件 重命名为 server

目录结构:

```
asset/
├── client/                 # 客户端美术资源
├── server/                 # 服务端配置文件
└── ...
```
